# Contributing to CampusFlow

Thank you for your interest in contributing!

## Branch Naming
- `feat/your-feature` for features
- `fix/bug-description` for bug fixes
- `chore/task-name` for maintenance

## Commit Style
We use conventional commits:
- `feat:`, `fix:`, `docs:`, `chore:`, `ci:`, `perf:`, `style:`

## Pull Requests
All PRs should target the `main` branch with a clear description.

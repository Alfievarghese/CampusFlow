# Changelog

All notable changes to CampusFlow are documented here.

## [1.0.0] - 2025-02-20

### Added
- Multi-admin role system with approval workflow
- Hall management with conflict detection
- Event booking with override request workflow
- Recurring events (weekly, monthly, custom)
- Public RSVP and invite-only flow
- FullCalendar integration
- Audit log system
- Supabase PostgreSQL via Prisma ORM
- JWT authentication with bcrypt
- Vercel deployment configuration
